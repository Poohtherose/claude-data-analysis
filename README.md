# SPSS ANOVA 统计分析平台

一个基于Web的科研数据统计分析工具，模拟IBM SPSS 27的分析流程，实现单因素ANOVA检验、LSD检验、Duncan检验和方差齐性检验。

## 功能特性

- **单因素方差分析 (One-Way ANOVA)** - 检验多组间均值差异的显著性
- **LSD检验 (Least Significant Difference)** - 事后多重比较
- **Duncan检验 (Duncan's Multiple Range Test)** - 邓肯多重范围检验
- **方差齐性检验 (Levene's Test)** - 检验各组方差是否相等
- **自动识别数据** - 智能识别Excel文件中的样品名称和指标数据
- **多指标分析** - 一次分析多个指标数据
- **SPSS风格报告** - 输出类似SPSS的专业统计分析报告

## 界面预览

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

或者使用启动脚本自动安装：

```bash
python start_server.py --install
```

### 2. 启动服务器

```bash
python start_server.py
```

或者使用Flask命令：

```bash
flask run
```

服务器将在 http://localhost:5000 启动

### 3. 生成测试数据（可选）

```bash
python generate_sample_data.py
```

这将生成两个示例Excel文件：
- `sample_data.xlsx` - 单指标数据
- `multi_indicator_sample.xlsx` - 多指标数据

## 使用指南

### 数据格式要求

上传的Excel文件应包含：
1. **样品名称列** - 标识不同组别的名称（如"对照组"、"处理组A"等）
2. **指标数据列** - 数值型的实验测量结果

示例数据格式：

| 样品名称 | 指标结果 |
|---------|---------|
| 对照组   | 50.2    |
| 对照组   | 48.5    |
| 处理组A  | 55.3    |
| 处理组A  | 54.1    |
| 处理组B  | 62.5    |
| 处理组B  | 61.8    |

### 操作步骤

1. **上传文件** - 点击上传区域或拖拽Excel文件到上传区域
2. **选择列** - 选择样品名称列和要分析的指标列
3. **开始分析** - 点击"开始分析"按钮执行统计检验
4. **查看结果** - 在页面下方查看ANOVA、方差齐性等分析结果
5. **下载报告** - 点击"下载完整报告"获取Excel格式的详细报告

### 报告内容

下载的Excel报告包含以下工作表：

- **描述性统计** - 各组的均值、标准差、标准误等
- **ANOVA表** - 方差分析结果（SS、df、MS、F、p值）
- **方差齐性检验** - Levene检验结果
- **LSD检验表** - 两两比较的均值差异、标准误、t值、p值
- **Duncan检验表** - 邓肯检验的临界值和显著性

## 统计方法说明

### 单因素ANOVA
检验多组独立样本的均值是否存在显著差异。
- H0: 所有组的均值相等
- H1: 至少有一组均值不同

### Levene方差齐性检验
检验各组方差是否相等，用于判断ANOVA的前提假设是否满足。
- p > 0.05: 方差齐性（满足ANOVA假设）
- p ≤ 0.05: 方差不齐

### LSD检验
当ANOVA显著时，进行两两比较确定哪些组之间存在差异。
- 计算每对组间的均值差异
- 与LSD临界值比较判断显著性

### Duncan检验
一种保守的多重比较方法，用于将各组划分为同质子集。
- 按均值排序并分配字母标记
- 相同字母表示组间无显著差异
